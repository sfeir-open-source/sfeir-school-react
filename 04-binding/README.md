# 04-binding instructions

## STEP 1

- Create a folder **feature/home** in the app folder
- Create a file **Home.tsx** and **Home.scss** inside the **feature/home** folder
- Copy and paste the content from the **assets/\_Home.html** and **assets/\_Home.scss** files into the **Home.tsx** and **Home.scss** files respectively
- Import the **Home.scss** file inside the **Home.tsx** file

## STEP 2

- Retrieve in the HomeComponent the firstPerson of the PEOPLE array (file available in the folder **mocks**)
- Realize all the bindings necessary to display the information of the first person in the HomeComponent

## STEP 3

- Call the Home component into the AppComponent inside a section with the class **people-application__pages**

## STEP 4

Please use the following command to check if your solution is working properly

```bash
npm run serve -- 04-binding
```
