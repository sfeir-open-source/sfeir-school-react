# Getting started

Install all the dependencies
```bash
npm ci
```
<!-- .element: class="big-code"-->

<br/>

Launch a workshop
```bash
npm run client -- {{ workshop_name }}
```
<!-- .element: class="big-code"-->

<br/>

Launch the server
```bash
npm run server:start 
```
<!-- .element: class="big-code"-->
