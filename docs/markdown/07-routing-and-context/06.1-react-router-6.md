# Main differences with v5

```jsx
// Change this:
<Switch>
  <Redirect from="about" to="about-us" />
</Switch>
```

##--##
<!-- .slide: class="with-code" -->

```jsx
// to this:
<Switch>
  <Route path="about" render={() => <Redirect to="about-us" />} />
</Switch>
```

# Route matching

```jsx
// Change this:
 <BrowserRouter>
      <Switch>
        <Route exact path="/">
          <Home />
        </Route>
        <Route path="/users">
          <Users />
        </Route>
      </Switch>
    </BrowserRouter>
```

##--##
<!-- .slide: class="with-code" -->

```jsx
// to this:
 <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="users/*" element={<Users />} />
      </Routes>
  </BrowserRouter>
```
<!-- .element: style="margin-top:200px" -->

No exact, use `path/*` to match
